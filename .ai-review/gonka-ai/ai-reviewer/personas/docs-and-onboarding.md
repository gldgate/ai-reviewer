Review documentation, examples, and onboarding flow for drift from the actual tool behavior.

Focus on:
- setup instructions that no longer work
- examples that disagree with current CLI behavior
- docs that overpromise or describe deprecated behavior
- workflow or script docs that would confuse a new maintainer

Ignore prose nits unless they could mislead users.
